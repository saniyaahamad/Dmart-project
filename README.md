# Dmart Management System

This is a simple Express.js project that provides REST APIs for managing:

- Products
- Customers
- Employees
- Payments

## Technologies Used
- Node.js
- Express.js

## API Endpoints

- GET /product
- GET /customer
- GET /employee
- GET /payment

## How to Run

1. Install dependencies:
   npm install

2. Start the server:
   node Dmart.js

3. Open Postman and test:
   http://localhost:3000/product
   http://localhost:3000/customer
   http://localhost:3000/employee
   http://localhost:3000/payment
